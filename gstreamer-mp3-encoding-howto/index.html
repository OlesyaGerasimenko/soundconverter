GStreamer MP3 Encoding mini-HowTo *

* or how to enable mp3 output in SoundConverter

Ubuntu

Ubuntu 7.10 to 11.4, click here.
Ubuntu 11.10, click here.

Mandriva

Add PLF sources with EasyUrpmi.
And then install package gstreamer0.10-lame.

Fedora

Add RPM Fusion repository.
Install gstreamer-plugins-ugly.

Gentoo

Install package gst-plugins-lame.

Debian

Add the deb-multimedia repository to apt sources.
Install package gstreamer0.10-plugins-ugly.

OpenSuse

You need the gstreamer-0_10-plugins-ugly-orig-addon package, available here: packman package.

Arch Linux

Install package gst-plugins-ugly.
